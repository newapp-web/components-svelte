# 欢迎使用蜂鸟组件库-anyJS版

组件库，是适用任何JS框架和纯JavaScript场景。该组件库维护和管理组件库的开发、调试、预览、部署过程。

组件使用svelteJS开发，编译成纯JS和CSS文件，以便于任意依赖使用。
